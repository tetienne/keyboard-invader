    vite.config.ts (verify Tailwind plugin and alias config from Plan 01)
    tsconfig.json (verify strict mode and paths from Plan 01)
    src/style.css (verify Tailwind @theme exists from Plan 01)
  </read_first>
  <action>
    **1. Create `index.html`** (Vite entry, per UI-SPEC HTML structure):
    ```html
    <!doctype html>
    <html lang="fr">
      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Keyboard Invader</title>
        <style>
          body {
            margin: 0;
            overflow: hidden;
            background: #1a1a2e;
          }
          #game-container {
            width: 100vw;
            height: 100vh;
          }
        </style>
      </head>
      <body>
        <div id="game-container"></div>
        <script type="module" src="/src/main.ts"></script>
      </body>
    </html>
    ```
    - Body background `#1a1a2e` prevents white flash before PixiJS init (per UI-SPEC)
    - `#game-container` is full viewport with no padding (per UI-SPEC)
    - No `<link>` to style.css needed -- main.ts will import it for Tailwind

    **2. Create `src/main.ts`** (per D-16, RESEARCH.md Pattern 5, UI-SPEC Canvas Test Page Layout):
    - Import `'./style.css'` at top (activates Tailwind for the page)
    - Import `{ Application, BitmapText, BitmapFont }` from `'pixi.js'`
    - Define an `async function init(): Promise<void>` (NEVER top-level await -- RESEARCH.md anti-pattern)
    - Inside `init()`:
      1. Create `const app = new Application()`
      2. Call `await app.init({ background: '#1a1a2e', resizeTo: window })`
      3. Get `document.getElementById('game-container')` -- if null, throw Error
      4. Append `app.canvas` to the container element
      5. Install BitmapFont: `BitmapFont.install({ name: 'GameFont', style: { fontFamily: 'Arial', fontSize: 48, fill: '#ffffff' } })`
      6. Create `const title = new BitmapText({ text: 'Keyboard Invader', style: { fontFamily: 'GameFont', fontSize: 48 } })`
      7. Center title: `title.x = app.screen.width / 2 - title.width / 2` and `title.y = app.screen.height / 2 - title.height / 2`
      8. `app.stage.addChild(title)`
    - Call `void init()` at module level

    Ensure all code passes ESLint strict + naming-convention. Use explicit types. No `any`.
  </action>
  <verify>
    <automated>pnpm exec tsc --noEmit && pnpm exec eslint src/main.ts && pnpm run build && test -f dist/index.html</automated>
  </verify>
  <done>index.html and main.ts exist, TypeScript compiles, ESLint passes, `pnpm run build` produces dist/index.html, PixiJS initializes with BitmapText title centered on dark background.</done>
