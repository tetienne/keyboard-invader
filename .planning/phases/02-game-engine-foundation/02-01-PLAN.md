**src/game/canvas.ts -- Pure scaling math (D-14, D-15):**

Export a pure function `computeScale(screenWidth: number, screenHeight: number): ScaleResult`. This is the letterbox calculation from RESEARCH Pattern 3, extracted as a pure function for testability.

```typescript
import type { ScaleResult } from './types.js'
import { BASE_WIDTH, BASE_HEIGHT } from './types.js'

export function computeScale(screenWidth: number, screenHeight: number): ScaleResult {
  const scale = Math.min(screenWidth / BASE_WIDTH, screenHeight / BASE_HEIGHT)
  const offsetX = (screenWidth - BASE_WIDTH * scale) / 2
  const offsetY = (screenHeight - BASE_HEIGHT * scale) / 2
  return { scale, offsetX, offsetY, screenWidth, screenHeight }
}
```

Also export `setupCanvas(app: Application, gameRoot: Container): () => void` that attaches a resize listener, applies scale/offset to gameRoot, resizes renderer, and returns a cleanup function. This function uses PixiJS types but the resize handler calls `computeScale` internally.

Also export `applyScale(gameRoot: Container, app: Application, result: ScaleResult): void` that sets `gameRoot.scale.set(result.scale)`, `gameRoot.x = result.offsetX`, `gameRoot.y = result.offsetY`, and calls `app.renderer.resize(result.screenWidth, result.screenHeight)`.
