    ```typescript
    export interface DifficultyParams {
      readonly fallSpeed: number
      readonly spawnInterval: number
      readonly complexityLevel: number  // 0 = base, 1 = +top/medium, 2 = +bottom/all
    }

    export interface DifficultyConfig {
      readonly mode: 'letters' | 'words'
      readonly minFallSpeed: number
      readonly maxFallSpeed: number
      readonly baseFallSpeed: number
      readonly minSpawnInterval: number
      readonly maxSpawnInterval: number
      readonly baseSpawnInterval: number
      readonly speedStep: number        // 5 px/s per D-09
      readonly spawnStep: number        // 150 ms per D-09
      readonly deadZoneLow: number      // 0.6 per D-07
      readonly deadZoneHigh: number     // 0.8 per D-07
      readonly windowSize: number       // 10 per D-05
      readonly complexityUpThreshold: number   // 0.8 per D-11/D-12
      readonly complexityDownThreshold: number // 0.5 per D-11/D-12
      readonly complexityUpCount: number       // 5 per D-11
      readonly maxComplexityLevel: number      // 2 for letters, 1 for words per D-12
    }
    ```

    **Class: DifficultyManager**
    - Constructor takes DifficultyConfig, initializes fallSpeed/spawnInterval to base values, complexityLevel to 0
    - `recordResult(hit: boolean)`: push to window array, shift if > windowSize, guard return if window not full (per D-05), compute accuracy, call adjust()
    - `adjust(accuracy)`: per D-07/D-08, if accuracy > deadZoneHigh: ramp (fallSpeed += speedStep, spawnInterval -= spawnStep), if accuracy < deadZoneLow: ease (fallSpeed -= speedStep*2, spawnInterval += spawnStep*2). Clamp all values per D-10. Call updateComplexity()
    - `updateComplexity(accuracy)`: track consecutiveHighAccuracy counter. If accuracy > complexityUpThreshold, increment counter; else reset to 0. If counter >= complexityUpCount and level < maxComplexityLevel, promote level and reset counter. If accuracy < complexityDownThreshold and level > 0, demote level
    - `get params()`: returns frozen DifficultyParams
    - `get rollingAccuracy()`: current window accuracy (0 if empty)
    - `get windowFull()`: window.length >= windowSize

    **Preset configs per D-02/D-03:**
