  <action>
    Create `src/game/progression.ts` as a pure logic module with zero PixiJS imports (per D-04).

    **Exports:**
    - `XpGain` interface: `{ baseXp: number, accuracyBonus: number, modeMultiplier: number, totalXp: number }`
    - `LevelUpResult` interface: `{ previousLevel: number, newLevel: number, levelsGained: number, remainingXp: number }`
    - `LEVEL_THRESHOLDS: readonly number[]` = `[0, 50, 120, 220, 360, 540, 780, 1080, 1440, 1900]`
      - Level 1: 0 XP, Level 2: 50 XP (~2 sessions), Level 3: 120 XP, ..., Level 10: 1900 XP (~8 sessions)
      - Per D-03: early levels fast (2-3 sessions), later levels slower (5-8 sessions)
    - `MAX_LEVEL = 10`
    - `calculateXpGain(hits: number, total: number, mode: GameMode): XpGain`
      - Base: 2 XP per hit (per D-01)
      - Accuracy bonus: `Math.round(baseXp * (hits/total) * 0.5)` -- higher accuracy earns more (per D-01)
      - Mode multiplier: 1.5 for words, 1.0 for letters (per D-02)
      - totalXp = `Math.round((baseXp + accuracyBonus) * modeMultiplier)`
      - Handle edge case: total === 0 returns all zeros
    - `resolveLevel(totalXp: number): number`
      - Iterates LEVEL_THRESHOLDS from end, returns first level where totalXp >= threshold
      - Falls back to 1
    - `applyXp(currentXp: number, currentLevel: number, xpGain: number): LevelUpResult`
      - newTotalXp = currentXp + xpGain
      - newLevel = Math.min(resolveLevel(newTotalXp), MAX_LEVEL)
      - Returns { previousLevel: currentLevel, newLevel, levelsGained: newLevel - currentLevel, remainingXp: newTotalXp }
    - `xpForCurrentLevel(totalXp: number, level: number): { current: number, required: number }`
      - current = totalXp - LEVEL_THRESHOLDS[level - 1]
      - required = LEVEL_THRESHOLDS[level] - LEVEL_THRESHOLDS[level - 1]
      - At max level (no next threshold): return { current: 0, required: 0 }

    Import `GameMode` from `../game/types.js`.

    Write tests FIRST (red), then implement (green). Cover all behavior items above.
    Group into describe blocks: "calculateXpGain", "resolveLevel", "applyXp", "xpForCurrentLevel".
