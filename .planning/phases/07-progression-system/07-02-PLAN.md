    **1. Create `src/game/xp-bar.ts`** -- a self-contained PixiJS Container component for the XP bar.

    The XpBar class manages its own Container with:
    - Background pill (dark rounded rect, color 0x2d2d4a)
    - Fill pill (colored rounded rect 0x6b8bf5, width proportional to progress)
    - Level text (BitmapText, left side)
    - Optional XP fraction text (BitmapText, right side, e.g. "45/120")
    - Optional XP earned text (BitmapText, above bar, accent color 0xe94560)

    ```typescript
    export interface XpBarConfig {
      width: number       // bar width in px
      height: number      // bar height in px
      showXpText: boolean // show "{current}/{required}" right of bar
      showEarnedText: boolean // show "+N XP" above bar
      fontSize: number    // BitmapText size for labels
    }

    export class XpBar {
      readonly container: Container
      // ...private fields for graphics, texts

      constructor(config: XpBarConfig)

      /** Set bar state immediately (no animation). Used for initial display and HUD. */
      setProgress(level: number, currentXp: number, requiredXp: number): void

      /** Set the "+N XP" earned text. Only visible if showEarnedText is true. */
      setEarnedText(xp: number): void

      /** Start animated fill from current to target progress. Returns duration in ms. */
      animateFill(fromProgress: number, toProgress: number, duration: number): void

      /** Call each frame during animation. Returns true when animation complete. */
      update(dt: number): boolean

      /** Reset fill to 0 (for level-up transitions). */
      resetFill(): void

      destroy(): void
    }
    ```

    **HUD variant** (per UI-SPEC):
    - `new XpBar({ width: 140, height: 10, showXpText: false, showEarnedText: false, fontSize: 18 })`
    - Position at x: 16, y: 24 (with level label "Niv. {n}" at x: 16, y: 4)

    **Results variant** (per UI-SPEC):
    - `new XpBar({ width: 320, height: 20, showXpText: true, showEarnedText: true, fontSize: 22 })`
    - Position centered at y: BASE_HEIGHT * 0.56

    **Animation mechanics:**
    - `animateFill` stores fromProgress, toProgress, elapsed=0, duration
    - `update(dt)` advances elapsed, computes progress with easeOutQuad: `t * (2 - t)` where t = elapsed/duration
    - Redraws the fill rect width as `barWidth * lerp(from, to, eased_t)`
    - Returns true when elapsed >= duration

    **Drawing approach:**
    - Use PixiJS Graphics with `g.roundRect(x, y, width, height, cornerRadius)` then `g.fill(color)`
    - Background: full width, cornerRadius = height/2 (pill shape per D-12)
    - Fill: variable width, same cornerRadius, redrawn on each progress change
    - Level text left-aligned at bar left edge, XP text right-aligned at bar right edge (per D-12)
