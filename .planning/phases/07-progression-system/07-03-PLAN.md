    **1. Create `src/game/celebration.ts`** -- full-screen celebration overlay (per D-05, D-06, D-07).

    ```typescript
    interface Particle {
      graphic: Graphics
      vx: number       // pixels/second
      vy: number       // pixels/second
      life: number     // 0-1, decreases over time
      maxLife: number   // seconds (2.0)
    }

    export class CelebrationOverlay {
      readonly container: Container
      private particles: Particle[] = []
      private levelText: BitmapText
      private backdrop: Graphics
      private elapsed = 0
      private readonly DURATION = 2500  // auto-dismiss after 2.5s
      private scalePhase = 0            // 0-600ms for bounce animation
      private done = false

      constructor(level: number)
      update(dt: number): boolean  // returns true when fully complete
      destroy(): void
    }
    ```

    **Constructor details:**
    - Create semi-transparent backdrop: full canvas (1280x720), fill 0x000000 at alpha 0.6
    - Create BitmapText for level: text from i18n key `progression.levelUp` with level interpolated, e.g. "Niveau 3 !"
      - FontFamily: 'GameFont', fontSize: 48, tint: 0xe94560 (accent color)
      - Anchor 0.5, position center (640, 360)
      - Initial scale: 0.0 (will animate up)
    - Spawn 40 particle Graphics (per UI-SPEC):
      - Each particle: small circle, radius random 3-6px
      - Random color from celebration palette: [0xff6b6b, 0x4ecdc4, 0xffe66d, 0xa78bfa, 0xfb923c, 0xf9a8d4]
      - Spawn position: center (640, 360)
      - Random velocity: angle = random 0-2PI, speed = random 100-300 px/s
      - `vx = Math.cos(angle) * speed`, `vy = Math.sin(angle) * speed`
      - life = 1.0, maxLife = 2.0
      - gravity = 120 px/s^2 downward (applied in update)

    **update(dt) details:**
    - dt is in milliseconds, convert to seconds for physics: `const ds = dt / 1000`
    - Advance elapsed by dt (ms)
    - **Scale animation** (0-600ms): `t = Math.min(scalePhase / 600, 1)`. Bounce easing:
      - If t < 0.6: scale = (t / 0.6) * 1.3 (grow to 1.3)
      - If t >= 0.6: scale = 1.3 - 0.3 * ((t - 0.6) / 0.4) (settle to 1.0)
      - Apply: `levelText.scale.set(scale)`
      - Advance scalePhase by dt
    - **Particle physics** (each particle):
      - `p.graphic.x += p.vx * ds`
      - `p.graphic.y += p.vy * ds`
      - `p.vy += 120 * ds` (gravity)
      - `p.life -= ds / p.maxLife`
      - `p.graphic.alpha = Math.max(0, p.life)`
    - **Auto-dismiss**: if elapsed >= 2500, set done = true, return true

    **2. Wire CelebrationOverlay into GameOverState**:

    Replace the placeholder celebrating phase from Plan 02:
    - In the `celebrating` phase of the resultsPhase state machine:
      - Create `new CelebrationOverlay(currentDisplayLevel + 1)` (the level being celebrated)
      - Add its container to the GameOverState container (on top of everything)
      - In update loop for this phase: call `celebration.update(dt)`. When it returns true:
        - Destroy celebration overlay
        - Decrement pendingLevelUps
        - Increment currentDisplayLevel
        - If pendingLevelUps > 0: reset XP bar, update level label, start next fill animation -> stay in celebrating cycle
        - If pendingLevelUps === 0: reset XP bar to new level, fill with remaining XP -> transition to 'done'
