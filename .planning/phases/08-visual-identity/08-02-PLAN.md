**1. Create `src/game/alien-container.ts`:**

```typescript
import { Container, Sprite, BitmapText, Texture, Assets } from 'pixi.js'
```

Class `AlienContainer extends Container`:
- Properties: `sprite: Sprite`, `label: BitmapText`, `bobPhase: number` (randomized 0 to 2*PI), `blinkTimer: number` (2000 + random * 3000 ms)
- Constructor takes `texture: Texture`, `letter: string`, `tint: number`:
  - Create Sprite from texture, anchor 0.5, add as child
  - Create BitmapText with fontFamily 'GameFont', fontSize 48 (letter sits on alien). Anchor 0.5. Tint from param. Add as child (on top of sprite).
- Method `updateIdle(dt: number)`: bobbing via `sin(bobPhase) * 3` on sprite.y offset, increment bobPhase by `dt * 0.003`. Blink timer countdown; when <= 0, briefly squash sprite scaleY to 0.7 for 100ms then restore to 1.0, reset timer to 2000 + random * 3000.
- Method `setLetter(letter: string, tint: number)`: update label text and tint.
- Method `setTexture(texture: Texture)`: update sprite texture.
- Static method `getRandomAlienTexture(wordMode: boolean): Texture`: uses Assets.get() to pick a random texture from ALIEN_TEXTURES_PATHS (letter mode) or WORD_ALIEN_TEXTURE_PATHS (word mode). These are already loaded by BootState.
- Method `reset()`: reset bobPhase, blinkTimer, sprite scale, label text, visible = false.

**2. Create `src/game/starfield.ts`:**

```typescript
import { Container, Graphics } from 'pixi.js'
import { BASE_WIDTH, BASE_HEIGHT } from './types.js'
import { SPACE_PALETTE } from './theme.js'
```

Class `Starfield`:
- Constructor takes `parent: Container`. Creates 3 layers, each is a Container added to parent.
- Layer configs: `[{ count: 40, speed: 10, alpha: 0.3, size: 1 }, { count: 25, speed: 25, alpha: 0.6, size: 2 }, { count: 15, speed: 45, alpha: 1.0, size: 3 }]`
- For each layer, pre-allocate star Graphics objects. Each star: `Graphics.circle(0, 0, size).fill(0xffffff)`. Random x in [0, BASE_WIDTH], random y in [0, BASE_HEIGHT]. Set alpha per layer config.
- Method `update(dt: number)`: For each star in each layer, move y += speed * dt / 1000. If y > BASE_HEIGHT, reset y to -size, randomize x.
- Method `setIntensity(level: number)`: Adjusts near layer alpha (0.8 + level * 0.04, capped at 1.0) and mid layer alpha (0.5 + level * 0.02, capped at 0.8). For D-15 difficulty visual shift.
- Method `destroy()`: Destroy all containers.

**3. Create `src/game/effects.ts`:**

```typescript
import { ParticleContainer, Particle, Texture, Assets, Container, Graphics } from 'pixi.js'
import { STAR_PARTICLE_PATH, SPACE_PALETTE } from './theme.js'
```

Class `DestructionEffect`:
- Holds a ParticleContainer with dynamicProperties: position, scale, alpha.
- Private array of active particle data: `{ p: Particle, vx: number, vy: number, life: number }[]`
- Constructor takes parent Container. Creates ParticleContainer, adds to parent.
- Method `burst(x: number, y: number, color: number, count = 12)`: Create `count` Particle objects from star texture (via `Assets.get(STAR_PARTICLE_PATH)`). Radial scatter with random speed 80-200. Tint matches the destroyed letter color. Life = 1.0.
- Method `update(dt: number)`: Move particles by velocity * dt/1000, fade alpha by life (life -= dt/1000 * 2), shrink scale. Remove dead particles (life <= 0) from ParticleContainer.
- Method `clear()`: Remove all active particles.

Class `LaserBolt`:
- Simple Graphics-based laser line. Constructor takes parent Container.
- Method `fire(fromX: number, fromY: number, toX: number, toY: number)`: Draw a 4px wide line from (fromX, fromY) to (toX, toY) in SPACE_PALETTE.accent color. Animate alpha from 1 to 0 over 200ms.
- Method `update(dt: number)`: returns true when bolt animation complete.
- Internal state tracks active bolt.
