**2. Rewrite `src/avatars/renderer.ts` (per D-10):**

Replace the Graphics-based drawAvatar with SVG Sprite rendering:

```typescript
import { Sprite, Assets, Texture, Container, Graphics } from 'pixi.js'
import type { AvatarDefinition } from './definitions.js'

export function drawAvatar(container: Container, def: AvatarDefinition, size: number): Sprite {
  // Load pre-cached SVG texture (loaded by BootState)
  const texture = Assets.get<Texture>(def.svgPath)
  const sprite = new Sprite(texture)
  sprite.width = size
  sprite.height = size
  sprite.anchor.set(0.5)
  container.addChild(sprite)
  return sprite
}
```

The signature changes: instead of `drawAvatar(g: Graphics, def, size)` it becomes `drawAvatar(container: Container, def, size): Sprite`. This requires updating all call sites in profile-state.ts and states.ts (GameOverState).

**Shared pattern: Space panel helper function**

At the top of states.ts (or in a small utility), create a helper for drawing themed panels used by menus:

```typescript
function drawSpacePanel(g: Graphics, x: number, y: number, w: number, h: number): void {
  // Background
  g.roundRect(x, y, w, h, UI_CONSTANTS.panelCornerRadius)
  g.fill({ color: SPACE_PALETTE.secondary, alpha: UI_CONSTANTS.panelBgAlpha })
  // Glow border
  g.roundRect(x, y, w, h, UI_CONSTANTS.panelCornerRadius)
  g.stroke({ color: SPACE_PALETTE.glow, width: UI_CONSTANTS.panelBorderWidth, alpha: UI_CONSTANTS.panelBorderAlpha })
}
```
