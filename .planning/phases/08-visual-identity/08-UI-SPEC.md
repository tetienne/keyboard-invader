# Phase 8 -- UI Design Contract

> Visual and interaction contract for the Visual Identity phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (PixiJS canvas game, not DOM component library) |
| Preset | not applicable |
| Component library | PixiJS v8.17.1 (Container, Sprite, BitmapText, ParticleContainer) |
| Icon library | none (custom SVG sprites for aliens, avatars, spaceship) |
| Font | Fredoka via @fontsource/fredoka 5.2.10 (rounded/bubbly, child-friendly) |

**Note:** This project renders on PixiJS canvas. Design tokens serve two contexts: (1) PixiJS in-canvas rendering (hex integer colors, pixel sizes in BitmapText), and (2) Tailwind CSS tokens in `src/style.css` for the HTML wrapper. Both must stay in sync.

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, alien label offset from sprite center |
| sm | 8px | Compact element spacing, star particle scatter radius |
| md | 16px | Default element spacing, button padding, panel inner padding |
| lg | 24px | Section padding, space between UI groups on menu screens |
| xl | 32px | Layout gaps, gap between avatar slots on profile screen |
| 2xl | 48px | Major section breaks, spaceship-to-play-area margin |
| 3xl | 64px | Page-level spacing, top/bottom safe zones for HUD elements |

Exceptions: Touch target minimum of 44px for avatar selection slots and menu buttons (accessibility baseline for children).

**Source:** Existing `src/style.css` tokens (no change needed, already defined).

---

## Typography

All text renders via PixiJS BitmapFont generated from Fredoka at boot time.

| Role | BitmapFont Size | Weight | Line Height | Usage |
|------|----------------|--------|-------------|-------|
| Body | 32px | 400 (Regular) | 1.5 | Score counter, session stats, menu descriptions |
| Label | 24px | 400 (Regular) | 1.4 | Button labels, avatar names, level titles, HUD text |
| Heading | 48px | 700 (Bold) | 1.2 | Screen titles ("Mode Selection", "Game Over"), level-up announcement |
| Display | 80px | 700 (Bold) | 1.1 | Falling letter inside alien container, large celebration text |

**BitmapFont generation:** Install at resolution 2 (fixes D-22 blurriness). Generate ASCII character set plus accented French characters (e, a, u, i, o with accents). Two BitmapFont instances: `GameFont` (400 weight) and `GameFontBold` (700 weight).

**Tailwind CSS tokens** in `src/style.css` update to match:
- `--font-size-body`: 16px (unchanged, for HTML elements only)
- `--font-size-label`: 14px (unchanged)
- `--font-size-heading`: 24px (unchanged)
- `--font-size-display`: 48px (unchanged)

**Source:** D-19 (rounded/bubbly font), D-22 (fix blurriness), RESEARCH.md Pattern 2.

## Visual Elements Contract

This section is specific to the PixiJS canvas game and replaces the standard "Component Inventory" section.

### Alien Creatures (D-02, D-03, D-07)

| Property | Value |
|----------|-------|
| Letter-mode designs | 6-8 SVG files, 64x64 to 128x128 viewBox |
| Word-mode designs | 2-3 SVG files, 128x128 viewBox (larger, multi-eyed) |
| Style | Round, blobby, soft shapes, big eyes, no sharp edges |
| Max SVG file size | 2KB per alien (simple paths, no gradients, no filters) |
| Letter placement | BitmapText centered on alien sprite (Display size 80px, tinted per letter palette) |
| Idle animation | Gentle bobbing: sin(phase) * 3px vertical, randomized start phase |
| Blink animation | Brief sprite squash every 2-5 seconds (randomized interval) |
| Hit reaction (D-12) | Surprised face swap, then 12 star/sparkle particles burst outward, 500ms total |
| Miss reaction (D-13) | Quick horizontal dodge (20px, 300ms) or wobble, returns to position |
| Escape animation (D-14) | Waves goodbye, accelerates downward with fading trail, 600ms |

### Defender Spaceship (D-17)

| Property | Value |
|----------|-------|
| SVG file | spaceship.svg, 64x64 viewBox |
| Position | Bottom center, 48px from canvas bottom edge |
| Shoot animation | Brief upward laser bolt (4px wide line, accent color #e94560, 200ms travel) |
| Idle | Subtle hover bob, same sin pattern as aliens but slower |

### Starfield Background (D-05)

| Property | Value |
|----------|-------|
| Layer count | 3 (far, mid, near) |
| Far layer | 40 stars, speed 10px/s, alpha 0.3, size 1px |
| Mid layer | 25 stars, speed 25px/s, alpha 0.6, size 2px |
| Near layer | 15 stars, speed 45px/s, alpha 1.0, size 3px |
| Star recycle | Stars that scroll off bottom reposition at top (no allocation) |
| Difficulty shift (D-15) | Stars brighten, nebula hue shifts toward lighter purple at higher difficulty |
| Optional decoration | Planet or moon in corner, static, low alpha (0.2), does not interfere with gameplay |

### Destruction Particles (D-12, D-16)

| Property | Value |
|----------|-------|
| Container | ParticleContainer (not regular Container) |
| Particle shape | Star-shaped SVG texture (star.svg, 16x16 viewBox) |
| Hit burst | 12 particles per hit, radial scatter, 500ms lifetime, fade to alpha 0 |
| Particle color | Matches the destroyed alien's letter tint color |
| Level-up particles | Stars, rockets, planet confetti shapes. Brief "warp speed" star streak (200ms). |

### UI Panels (D-21)

| Property | Value |
|----------|-------|
| Corner radius | 12px |
| Border | 2px solid, glow color #6b8bf5 at alpha 0.6 |
| Background | Secondary color #2d1b4e at alpha 0.9 |
| Glow effect | Outer glow via PixiJS filters or double-border trick (2px blur spread) |
| Button style | Rounded rectangle, accent fill for primary, secondary fill for non-primary, glow border |

### Avatar Art (D-08, D-09, D-10)

| Property | Value |
