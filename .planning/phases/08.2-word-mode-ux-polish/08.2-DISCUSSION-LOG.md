## Per-Character Feedback

| Option | Description | Selected |
|--------|-------------|----------|
| Color change | Typed chars turn gold, untyped stay white. Simple, clear, consistent with Phase 3 hit feedback. | ✓ |
| Color + pulse animation | Typed chars turn green AND briefly scale up (tiny pop). More rewarding but adds animation complexity. | |
| Progressive glow | Typed chars get bright tint, untyped stay dimmed/semi-transparent. "Lighting up" effect. | |

**User's choice:** Color change (Recommended)
**Notes:** Simple approach preferred — no per-character animation.

### Typed Character Color

| Option | Description | Selected |
|--------|-------------|----------|
| Green (#44FF44) | Classic "correct" signal. Matches letter mode hit feedback. | |
| Gold/yellow (#FFD700) | Feels like "progress" / XP gain. Warm, distinct from alien body colors. | ✓ |
| You decide | Claude picks a color. | |

**User's choice:** Gold/yellow (#FFD700)
**Notes:** Gold chosen for its "progress" feel, distinct from letter mode green.

### Cursor Indicator

| Option | Description | Selected |
|--------|-------------|----------|
| Underline cursor | Small underline under next character. Common in typing tutors. | |
| No cursor — just color split | Gold (typed) vs white (untyped) boundary serves as cursor. Simpler. | ✓ |
| Blinking/pulsing character | Next character gently pulses. More noticeable but potentially distracting. | |

**User's choice:** No cursor — just color split
**Notes:** The gold/white boundary IS the cursor. Keeps the word clean.
